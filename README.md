# Mathematical Methods for Physics: A Comprehensive Guide

## 📌 About This Project
This is a comprehensive set of lecture notes and tutorials I independently authored during my high school years (prior to joining Imperial College London). 

Driven by a deep passion for theoretical physics, I systematically compiled, derived, and structured advanced mathematical tools required for university-level physics.

## 🧠 Core Topics Covered
* Partial Differential Equations in Physics
* Linear Algebra & Vector Calculus
* Fourier & Laplace Transforms

## 🎯 Why This Matters for Engineering
While this is a purely theoretical document, the process of writing it trained my ability to:
1. **Systematize Complex Logic:** Distill highly abstract mathematical concepts into digestible, standard operating procedures (SOPs).
2. **First-Principles Thinking:** Understand the absolute mathematical foundation behind physical signals, which is critical for signal processing, modeling, and metrology in real-world engineering.
